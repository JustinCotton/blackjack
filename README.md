# Blackjack Card Game Project
## Overview



**Live site:** <https://justincotton.github.io/blackjack/>

## Technologies Used

  * Languages: HTML5, CSS3, Javascript, jQuery
  * Design: Google Fonts
  * Project Planning & User Stories: [Trello]()

  * Git, Github, Github Pages

  * Wagering
